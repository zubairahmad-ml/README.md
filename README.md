## 16. Adding Tables
Tables banane ke liye `|` aur dashes `-` ka istemal hota hai.

| Species | Petal | Sepal |
| :--- | :---: | ---: |
| AA | 22 | 33 |

> - `:---:` se center align hota hai.

## 18. Install Extensions & Shortcuts

## 19. Making Links via Right Click
1. Pehle text likhein (e.g., "My YouTube Link").
2. Usay select karein aur **Right Click** karein.
3. **Toggle Hyperlink** par click karein.
4. Upar box mein URL paste karke **Enter** dabayein.

---

## 20. Adding Images via Right Click
1. Image ka naam likhein.
2. Select karke **Right Click** karein.
3. **Toggle Image** select karein aur image file ka naam (e.g., `map.jpg`) likh dein.
> **Note:** Agar `!` remove kar dein toh yeh sirf link ban jayega, image nahi dikhegi.

## 24. Make PDF & Export
Markdown file ko PDF banane ke liye:
1. `Markdown PDF` extension install karein.
2. File par **Right Click** karein aur **Markdown PDF: Export (pdf)** select karein.
3. Aap ise **HTML** ya **JPG** mein bhi export kar sakte hain.
